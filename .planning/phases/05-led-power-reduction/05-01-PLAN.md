<objective>
Reduce LED power consumption by halving brightness (64->32) and halving all blink durations.

Purpose: Each wake cycle wastes energy on NeoPixel current draw and time spent blinking. Halving brightness cuts NeoPixel current roughly in half; halving blink durations reduces active time by ~400ms per cycle.

Output: Updated main.py with LED_BRIGHTNESS constant, reduced default brightness, and halved blink timings. Updated tests confirming new defaults.
</objective>

# Module-level constant to add:
# LED_BRIGHTNESS = 32  (new, for Phase 7 configurability)

class StatusLED:
    def __init__(self, pin_num=27, brightness=64):  # Change default to LED_BRIGHTNESS (32)

    def _scale(self, val):
        return min(255, int(val * self.brightness / 255))

    # Blink defaults to halve (current -> new):
