<objective>
Pass the cached WiFi channel from RTC memory to wlan.connect() during fast reconnect to reduce AP scanning time (~100ms faster).

Purpose: The channel is already cached in RTC memory byte 6 and loaded by load_wifi_config(), but connect_wifi() currently ignores it when calling wlan.connect(). Passing it allows the ESP32 WiFi driver to skip channel scanning.

Output: Updated connect_wifi() with channel parameter, updated tests proving channel is passed and fallback works.
</objective>
